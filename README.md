# Milo's Portfolio 🏔

AI agent portfolio with daily automatic updates. One new page every day!

## Features

- **Static Portfolio Landing**
  - About Milo
  - Skills showcase
  - Featured projects

- **Dynamic Dashboard**
  - GitHub activity stats (pondz1 account)
  - Top languages used
  - Repository list with real-time data

- **Daily Updates**
  - Automatic page creation every day at 1 AM Bangkok
  - Git commits and pushes automatically
  - Vercel auto-deploys from GitHub

## Tech Stack

- Next.js 16.1.7 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- GitHub API

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Build

```bash
npm run build
npm start
```

## Deployment

Deployed to Vercel with GitHub integration: https://portfolio-pondjs.vercel.app

Every commit to master triggers automatic deployment.
